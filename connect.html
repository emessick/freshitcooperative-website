Connect or Reconnect

This page is part of FITC's internal application infrastructure. It is reached during authorization flows for connecting business systems on behalf of Fresh IT Cooperative LLC.

What This Is

FITC's internal Chief of Staff application connects to authorized business systems (QuickBooks Online, Microsoft 365, FreshDesk, Restaurant365, Ramp, banking, and others) using OAuth 2.0. When an FITC administrator initiates or renews a connection, this page is the public-facing endpoint referenced in our application registrations with each provider.

Connecting

Active OAuth flows for FITC's connected applications are completed by an authorized administrator using FITC-controlled credentials. The actual authorization callback is handled by an internal service running on FITC infrastructure; this page is informational and does not perform authentication itself.

Reconnecting

If a previous connection has been disconnected, expired, or the credentials have been rotated, an FITC administrator can re-initiate the OAuth flow through the relevant provider (e.g., for QuickBooks Online, via app.qbo.intuit.com) with the same flow used for the initial connection.
